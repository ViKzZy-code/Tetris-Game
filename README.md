# 🧩 Jogo de Tetris

Um jogo de **Tetris** desenvolvido com **HTML**, **CSS** e **JavaScript**, recriando um dos maiores clássicos dos videogames de forma simples, interativa e responsiva. O projeto utiliza lógica de programação para movimentação e rotação das peças, detecção de colisões, remoção de linhas completas e gerenciamento da pontuação, proporcionando uma experiência fiel ao jogo original.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6)

## Funcionalidades

* Movimentação das peças para a esquerda, direita e para baixo.
* Rotação das peças em diferentes direções.
* Geração aleatória das peças (Tetrominós).
* Detecção de colisões com as bordas e peças já posicionadas.
* Remoção automática de linhas completas.
* Sistema de pontuação baseado nas linhas eliminadas.
* Aumento gradual da velocidade conforme o progresso da partida.
* Reinício do jogo após o fim da partida (Game Over).

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web, lógica de programação, estruturas de dados, manipulação do DOM e eventos utilizando JavaScript puro, além de reforçar boas práticas na construção de aplicações interativas.
